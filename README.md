# weather-app

## About
It's a weather application written in **TypeScript** that fetches data from an **API**. In the project, **Tailwind CSS** is also used.

## Installation
`git clone https://github.com/ivan-chukhalo/weather-app.git`<br>
`cd  weather-app`<br>
`npm i`

### Run demo
`npm run start`

## The project employs
- TypeScript
- API
- Tailwind CSS
